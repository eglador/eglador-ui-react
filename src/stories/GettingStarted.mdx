# Getting Started

> **Status:** Alpha — 55/55 components shipped.

## What's ready

**Infrastructure:**

- Build pipeline (`tsup` → CJS + ESM + DTS)
- TypeScript strict mode (`tsc --noEmit`)
- Tailwind v4 + `@source` integration
- Storybook 10 (`@storybook/addon-docs`, `@storybook/addon-a11y`, `@storybook/addon-vitest`)
- CI / npm publish workflows
- Shared floating-element positioning hook (`useFloating`) with viewport collision detection
- Portal-rendered overlays (Tooltip, Popover, HoverCard, Dropdown, ContextMenu, Dialog, AlertDialog, Drawer, Notification, Select, MultiSelect, Menubar, Calendar popovers)

**Layout & Structure (7/7):** Accordion · AspectRatio · Collapsible · Resizable · ScrollArea · Separator · Sidebar

**Forms (15/15):** Button · ButtonGroup · Checkbox · CheckboxGroup · Input · InputGroup · InputOTP · Label · MultiSelect · NativeSelect · Radio · RadioGroup · Select · Switch · Textarea

**Date & Time (3/3):** Calendar · DatePicker · DateTimePicker

**Overlays (10/10):** Alert · AlertDialog · ContextMenu · Dialog · Drawer · Dropdown · HoverCard · Notification · Popover · Tooltip

**Data (1/1):** Table

**Misc (5/5):** Command · ImageCropper · Link · Progress · SpeedDial

## Shared vocabulary

**Variants** (visual hierarchy): `solid` (filled high-contrast) · `soft` (filled muted) · `outline` (bordered) · `ghost` (transparent) · `link` (inline text)

**Sizes:** `xs` · `sm` · `md` · `lg` · `xl`

**Shapes** (where applicable): `square` · `rounded` · `pill` · `circle`

**Shadows** (where applicable): `none` · `xs` · `sm` · `md` · `lg` · `xl`

**Palette:** zinc-only — `text-zinc-{50→900}`, `bg-white`, `bg-zinc-{50→900}`, `border-zinc-{200→900}`. No color schemes (primary / danger / success etc.); theming is handled separately.

### Toast notifications

```tsx
import {
  NotificationContainer,
  useNotification,
  renderNotification,
  Button,
} from "eglador-ui-react";
import { CheckCircle2 } from "lucide-react";

function MyPage() {
  const { notifications, push, dismiss } = useNotification();
  return (
    <>
      <Button
        onClick={() =>
          push({
            icon: <CheckCircle2 />,
            title: "Saved",
            description: "Your changes are live.",
            duration: 4000,
          })
        }
      >
        Show toast
      </Button>
      <NotificationContainer position="top-right">
        {notifications.map((item) =>
          renderNotification({ item, onDismiss: dismiss })
        )}
      </NotificationContainer>
    </>
  );
}
```

## Design principles

- **Headless-first** — native HTML semantics are the foundation; styling is opt-in via Tailwind classes
- **Compound API** — Group / Item / Addon / Trigger / Content parts wired via context
- **A11y** — `role`, `aria-*`, focus-visible ring, `<fieldset>` / `<legend>` / `<label htmlFor>`, keyboard navigation
- **TypeScript-first** — generic props, strict mode, discriminated unions where helpful (Accordion, Calendar)
- **Tailwind v4** — `zinc` palette, logical CSS (`ps-*`, `me-*`, `border-s`, RTL automatic)
- **Zero runtime deps** — `clsx` and `tailwind-merge` are pre-bundled; portal positioning, focus traps, keyboard nav all implemented in-house
- **Controlled + Uncontrolled** — every stateful component supports both patterns
- **`asChild` projection** — most triggers and close buttons project props onto a single child via `cloneElement` so you can wrap Router `<Link>`s, custom Buttons, etc.
