# Getting Started

> **Status:** Pre-alpha — 10/55 bileşen hazır. Tier 2 (form bileşenleri) sürüyor.

---

## Hazır olan

**Altyapı:**

- Build pipeline (`tsup` → CJS + ESM + DTS)
- TypeScript strict mode (`tsc --noEmit`)
- Tailwind v4 + `@source` entegrasyonu
- Storybook 10 (`@storybook/addon-docs`, `@storybook/addon-a11y`, `@storybook/addon-vitest`)
- CI / npm publish workflows
- LICENSE, README, package metadata

**Tier 1 bileşenler (10/10):**

- `<AspectRatio>` — preset + numeric ratio
- `<Avatar>` + `<AvatarGroup>` — src/initials/icon fallback, +N overflow
- `<Badge>` — solid/soft/outline + removable
- `<Empty>` — empty state slot'lar
- `<Kbd>` + `<KbdGroup>` — klavye kısayolu
- `<Label>` — form label, required indicator
- `<Separator>` — yatay/dikey divider, label slot
- `<Skeleton>` — text/circular/rectangular/rounded + pulse/wave animasyonu
- `<Spinner>` — yükleniyor göstergesi
- `<Typography>` — h1-h4, p, lead, blockquote, list, code, kbd; RTL safe

Tüm bileşenler:
- `xs/sm/md/lg/xl` size standardı (size prop'u olanlar)
- `React.forwardRef` + `React.HTMLAttributes` spread
- Zinc-only paleti
- `rounded-sm` paket konvansiyonu

## Sıradaki — Tier 2

Basit state'li form bileşenleri:

Button · ButtonGroup · Link · Input · InputGroup · Textarea · Switch · Checkbox · CheckboxGroup · Radio · RadioGroup · Progress

---

## Tasarım prensipleri

- **Headless öncelikli** — her bileşenin bir hook karşılığı; render kullanıcıya
- **Compound API** — `<X.Root>`, `<X.Header>`, `<X.Body>` vb. tam markup kontrolü
- **A11y** — keyboard navigation, ARIA roles, focus management
- **TypeScript-first** — generic prop'lar, strict mode
- **Tailwind v4** — `zinc` paleti, `rounded-sm`, modern utility-first
- **Zero runtime deps** — `clsx` ve `tailwind-merge` pre-bundled
