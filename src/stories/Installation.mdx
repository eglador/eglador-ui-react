# Installation

Package installation, peer dependencies, and Tailwind v4 integration.

---

## 1. Install the package

```bash
npm install eglador-ui-react
```

```bash
yarn add eglador-ui-react
```

```bash
pnpm add eglador-ui-react
```

---

## 3. Add `@source` for Tailwind v4

Tailwind needs to scan classes inside the package. Add the `@source` directive to your global stylesheet:

```css
@import "tailwindcss";

@source "../node_modules/eglador-ui-react";
```

The `@source` path depends on your CSS file location:

## 4. Server components note (Next.js)

Components that use state (`useState` / `useEffect`) are marked `"use client"`. Place them inside a client component, or import them from a file that already has the `"use client"` directive at the top.
