# Installation

Paket kurulumu, peer dependency'ler ve Tailwind v4 entegrasyonu.

---

## 1. Paketi kur

```bash
npm install eglador-ui-react
```

```bash
yarn add eglador-ui-react
```

```bash
pnpm add eglador-ui-react
```

---

## 3. Tailwind v4 için `@source` ekle

```css
@import "tailwindcss";

@source "../node_modules/eglador-ui-react";
```

`@source` yolu CSS dosyanızın konumuna göre değişir:

## 4. Server components notu (Next.js)

Bileşenler `"use client"` ile işaretlidir. Onları bir client component içine veya `"use client"` direktifi olan bir dosyaya yerleştirin.
