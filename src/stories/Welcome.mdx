Headless, accessible component library for React — compound subcomponents, Tailwind v4, zero runtime dependencies, TypeScript-first.

> **Status:** Alpha — 55/55 components shipped.

---

## Installation

```bash
npm install eglador-ui-react
```

## Setup

```css
@import "tailwindcss";
@source "../node_modules/eglador-ui-react";
```

---

## Conventions

All components share the same vocabulary:

**Variants** (visual hierarchy, where applicable): `solid` · `soft` · `outline` · `ghost` · `link`

**Sizes** (where applicable): `xs` · `sm` · `md` · `lg` · `xl`

**Shapes** (where applicable): `square` · `rounded` · `pill` · `circle`

**Palette:** zinc-only (no color scheme; theming is a separate concern, handled later)

**Compound API:** Group / Item compound, context-based state, `<fieldset>` / `<legend>` semantics

**A11y:** Native HTML inputs, `aria-*` attributes, focus-visible rings, RTL-safe Tailwind logical properties
