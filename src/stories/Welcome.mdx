Eglador UI for React — headless, accessible component library with compound subcomponents, Tailwind v4, and zero runtime dependencies.

> **Status:** Pre-alpha — scaffold ready (build, Storybook, CI). 55 bileşen geliştirme sürecinde.

---

## Installation

```bash
npm install eglador-ui-react
```

## Setup

```css
@import "tailwindcss";
@source "../node_modules/eglador-ui-react";
```
